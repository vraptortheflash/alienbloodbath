**Android rewrite of the original Windows game "Alien Blood Bath".** This game is open sourced under the GPL version 3. Source code may be downloaded from this page using Subversion. For questions, please see the [FAQ](http://code.google.com/p/alienbloodbath/wiki/FAQ) or the [mailing list](http://groups.google.com/group/alien-blood-bath-developers).
